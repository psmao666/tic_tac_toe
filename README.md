# tic_tac_toe

## Introduction
This a simple tic_tac_toe game that supports two players (A, B).
The connection will be established via TCP/IP.

## How to set up

First, to use CMake, run
```
mkdir build
cd build
cmake ..
make
```

To run server, type
```
./TicTacToe_Server
```

To run client, type
```
./TicTacToe_Client
```
